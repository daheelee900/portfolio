4.블로그(리액트)

할일이나 아무글이나 포스팅하는 개인블로그
개발인원 : 1명

주요기능 : CRUD/페이징/리액트/리액트hook/async비동기처리/상태관리

프로젝트시연영상

배운점

자바스크립트를 배우고 그 외에 리액트 라이브러리로 간단한 블로그 프로그램을 만들어 보았다. 스스로 상태를 관리하는 캡슐화된 컴포넌트로 관리하는 개념이 매우 신비로웠다. react hook을 사용하여 상태관리인 usestate와 useeffect로 가변적인 상태를 쉽게 만들 수 있었다. 그리고 서버를 두개를 돌아가게 만들어 백사이트 서버로 db에 접근해서 필요한 데이터를 쉽게 가지고 왔다. 이 프로그램에서는 아직 redux를 사용하지 않았지만 앞으로의 프로젝트에서는 이 기술도 사용해봐야겠다.

사용기술 :

react

개발환경 :

node.js

개발도구 :

VSCode

DB :

mysql
